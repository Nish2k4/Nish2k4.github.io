# Portfolio — Minimal Mistakes (GitHub Pages)

A polished, mobile-friendly portfolio using the Minimal Mistakes theme.

## Quick Start
1. Create a **public** repo named `YOUR-USERNAME.github.io`.
2. Download this folder as ZIP and unzip.
3. Replace `YOUR-USERNAME` in `_config.yml` and links.
4. Commit & push to `main`.
5. In **Settings → Pages**, choose **GitHub Actions** for builds (recommended).

## Customize
- Change `minimal_mistakes_skin` to `dark`, `mint`, `sunrise`, etc.
- Replace `/assets/img/*` with your hero and project images.
- Edit files in `/projects` to add more work (each Markdown file = one project).

## Notes
- Remote theme loads automatically on GitHub Pages; no Gemfile needed.
- For custom domain, add a `CNAME` file and DNS record.
